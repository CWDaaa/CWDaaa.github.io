快速幂

今天做到了hdoj2035,这题要用到快速幂就去学习了一下简单的一些快速幂，好像还有有关于位运算的并没有去学习，因为我对位运算并不是很清楚

原理

如果普通的x的y次方我们的算法就是

   for(i=1;i<=y;i++)
   {
        x*=y;
    }

至于有math.pow（a,b）我并不是很喜欢用，而且在做oj上这题的时候并不适合

这种普通的for循环的时间复杂度是O（N），但是用了二分法的快速幂可以减到O（log2N），ACM就是要越快越好，所以学它！

二分法的原理很简单，这里我们拿2的1000次来讲解，大家都知道2^1000=（2×2）^500，这样时间复杂度就减少了一半，而4^500=（4×4）^250，这样我们的时间复杂度又少了一半，（16）^250=256^125，这里出现了125次，也很容易解决，256^125=（256×256）^62×256，后面都可以以此类推

没有看懂的小伙伴可以看下代码，这里方便Java选手看，就加了==0和!=0这种,其实C是不用的

递归法

   unsigned long pow(int x, unsigned y)
   {
     unsigned long tmp;
     if(y == 0) return 1;
     tmp = pow(x, y / 2);
     if（y % 2 == 0） return (tmp * tmp);
     else return (tmp * tmp * x);
    }

循环法

   int r=1;
   while(b!=0){
     if(b%2==1) r*=a;
     a*=a;
     b/=2;
   }

模运算

这里先要讲一下关于模运算的事，来来学姐提供的资料，a mod m == a % m

这里有几个公式
（a+b）mod m=（（a mod m）+（b mod m））mod m
（a-b）mod m=（（a mod m）-（b mod m））mod m
（a×b）mod m=（（a mod m）×（b mod m））mod m
除法是不可以的！！！
（a/b）mod m 是不等于（（a mod m）/（b mod m）） mod m
举个栗子，（100/50） mod 20=2，（100 mod 20）/（50 mod 20）mod 20=0，两个是不一样的
除法的取模需要逆元，想更加了解模运算的可以去查资料深入研究，这里就不再讨论了

HDOJ2035题解

刚刚为什么我们要讲模运算，因为这题就需要用到，我们可以看到a^b，a和b的最大值都是10000，求得的值也会很大，普通算法和快速幂都会很麻烦，但题目只要后三位，所以我们可以对快速幂进行一些优化，这里要用到（a×b）mod m=（（a mod m）×（b mod m））mod m，对每个递归或者循环出来的那个累乘数都取余

递归法

   import java.util.Scanner;

   public class HDOJ2035 {
       static  long qpow(int a, int n)
       {
           if  (n == 0)
               return 1;
           else if (n % 2 == 1)
               return qpow(a, n - 1) * a %1000;
           else
           {
               long temp = qpow((a*a)%1000, n / 2)%1000;
               return temp ;
           }
       }
       public static void main(String[] args) {
           Scanner kb=new Scanner(System.in);
           int a=kb.nextInt();
           int b=kb.nextInt();
           while(a!=0&&b!=0){
               System.out.println(qpow(a%1000,b)%1000);
               a=kb.nextInt();
               b=kb.nextInt();
           }
       }
   }

循环法

   #include <stdio.h>

   int main(void)
   {
       int m,n,b=1;
       while(~scanf("%d%d",&m,&n))
       {
           b=1;
           if(m==0&&n==0) break;
           for(int i=0;i<n;i++)
           {
               b*=m;
               b%=1000;
           }
           printf("%d\n",b); 
       }
       return 0;
   }
